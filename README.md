# homebox-inventory

Aplicación web ligera en HTML, CSS y JavaScript que transforma un CSV exportado de Homebox en un PDF listo para imprimir. La cabecera del PDF muestra cada ubicación y, debajo, una lista de items con su cantidad.

## Uso rápido
1. Abre `index.html` en tu navegador.
2. Selecciona el archivo CSV exportado (debe incluir las columnas `HB.location`, `HB.name` y `HB.quantity`).
3. Revisa la vista previa por ubicación.
4. Presiona **Generar PDF** y elige "Guardar como PDF" en el diálogo de impresión.
